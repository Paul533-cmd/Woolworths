# Woolworths Website Project

## Student Information

Name: Paul Katlego Monama

Module: Web Development

## Project Overview

This website was developed as part of the Web Development Portfolio of Evidence.

The website promotes Woolworths South Africa and provides information about the organisation, products, contact details and customer enquiries.

The website includes responsive design, JavaScript functionality and search engine optimisation techniques.

## Off-Page SEO Strategy

The website would be promoted through:

- Woolworths Facebook page
- Woolworths Instagram page
- Woolworths X (Twitter) page
- Woolworths LinkedIn page

Local SEO would be supported through:

- Store location pages
- Interactive maps
- Accurate contact information

Backlinks could be obtained through:

- Business directories
- Retail partnerships
- Social media profiles

## Page Speed Optimisation

The website was optimised by:

- Using external CSS and JavaScript files.
- Compressing images before upload.
- Using descriptive image names.
- Implementing lazy loading on images.
- Reducing unnecessary code.
